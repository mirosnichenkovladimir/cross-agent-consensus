# Final Report Prompt

Produce a human-readable final report.

Required sections:

1. Task
2. Participants
3. Artifact versions
4. Review Scope and Review Batch modes used
5. Findings summary by blocking/scope status
6. Accepted/fixed/verified in-scope blocking findings
7. Rejected findings and whether rejection was accepted
8. Non-blocking, deferred, and out-of-scope suggestions
9. Disputed or escalated in-scope blocking findings
10. Validation evidence
11. Terminal outcome for the declared scope
12. Human decision needed, if any
